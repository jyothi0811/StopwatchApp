# Stopwatch App

A simple Stopwatch Android application built using Kotlin and Jetpack Compose.

## Features
- Start timer
- Pause timer
- Reset timer
- Clean UI
- Real-time stopwatch updates

## Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
